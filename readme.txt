== Description ==

Themepaw Companion is a companion plugin for WordPress themes by themepaw.com. This plugin will work with other theme as well but it's originally focused to extend <strong><a href="https://themepaw.com/" target="_blank">themepaw.com</a> themes.</strong>

Elementor Page Builder plugin is mandatory for this plugin to work. This plugin will help you to create some advanced element with Elementor.

== Installation ==

This section describes how to install the plugin and get it working.
1. Go to "Add new" from plugins menu in your dashboard.
2. Search "Themepaw Companion" from the search section.
3. Click "Install Now" then "Activate" to make this plugin install and activate properly.
or
1. Upload the whole contents of the folder `themepaw-companion` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress dashboard
3. Enjoy using it :)

<strong>Note:</strong> Elementor Page Builder plugin is mandatory for this plugin to work

== Frequently Asked Questions ==

= Is this a standalone Plugin? =

No. You cannot use Themepaw Companion without Elementor as it\’s an addons plugin for Elementor.

= Does it work with any WordPress theme? =

Yes, it will work with any WordPress theme as long as you are using Elementor as a page builder.
